## Instrucciones

### 1. Clonar el repositorio:

```bash
git clone https://github.com/FireSouls25/Inferaz-form.git
```

### 2. Instalar paquetes:

```bash
cd Interfaz-form
```

```bash
npm install
```

### 3. Iniciar la pagina:

```bash
npm run dev
```

### 4. Abrir [http://localhost:3000](http://localhost:3000)
